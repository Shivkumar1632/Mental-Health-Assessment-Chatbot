# Mental Health Assessment Chatbot

## Project Overview
An AI-powered Mental Health Assessment Chatbot designed to provide mental health screening, emotional support, and personalized recommendations using Natural Language Processing (NLP) and Machine Learning techniques.

## Features
- Mental health assessment through interactive conversations
- AI-powered chatbot support
- User-friendly web interface
- Personalized responses and recommendations
- Secure user interaction

## Technologies Used
- HTML
- CSS
- JavaScript
- Python
- Flask
- Machine Learning
- NLP

## Future Enhancements
- Integration with advanced LLMs
- Real-time mental health monitoring
- Appointment booking with counselors
- Sentiment analysis dashboard
